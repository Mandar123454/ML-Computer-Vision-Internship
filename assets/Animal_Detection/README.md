# Animal_Detection assets

Put small sample media here to validate the notebook/app.

Accepted formats:
- Images: .jpg, .jpeg, .png
- Video: .mp4, .mov

Suggested minimum:
- 2–3 animal photos (dog/cat/bird)
- 1 short 3–5s video clip (zoo/park)

Example names:
- images/animal_01.jpg
- images/animal_02.jpg
- videos/zoo_short.mp4

Tips:
- Avoid heavy motion blur.
- Keep subjects reasonably centered and well lit.
