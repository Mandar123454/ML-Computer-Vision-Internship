# assets

Sample media for quick testing of each project. Keep files small (<10MB) and preferably royalty-free.

Projects:
- Animal_Detection: a few animal photos and one short video clip.
- Attendance_System: face reference images and webcam snapshots; includes an attendance CSV template.
- CarColor_Detection: car images from different angles; includes a sample color list.
- Drowsiness_Detection: eye-region frames and/or a short driving clip.
- Nationality_Detection: portrait photos; neutral background preferred.
- SignLanguage_Detection: short gesture clips or ordered image sequences.
